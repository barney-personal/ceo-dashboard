# Agent Instructions

**See [CLAUDE.md](./CLAUDE.md).** That file is the canonical source of truth for
agent guidance in this repository — architecture, conventions, secrets handling,
git workflow, and model/tool configuration.

Keeping agent guidance in a single file avoids drift. This repo is developed
with Claude Code; the `CLAUDE.md` filename is picked up automatically by that
tool, and other AI assistants should read the same file.
